🛠 ROOT CAUSE SUMMARY

All entries in COMDTY_DATA[] were initialized with px:null and relied 100% on fetchStooq() to populate them. When Stooq's CSV endpoint is blocked (CORS, rate-limit, or geo-block), all commodity prices stay null indefinitely. The chart header (At/Op/Hi/Lo/Vol) reads from window._chigh, window._clow, window._firstBbgPrice, window._cvol — these globals are only set by the TradingView iframe postMessage handler. For commodity symbols like BRENT (TVC:UKOIL), the iframe loads but often doesn't trigger the expected price-update messages in time. The patch adds Yahoo Finance futures as a parallel data layer and seeds the globals directly from real data.
